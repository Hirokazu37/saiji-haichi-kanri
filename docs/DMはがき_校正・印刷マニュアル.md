# DMはがき 校正・印刷マニュアル

DMはがきを「文面づくり → 校正（百貨店へ提出）→ 宛名印刷」まで進める手順書です。
入口：左メニュー **「DMハガキ」**（一覧 `/dm`）。

> **前提**：はがきは **裏面（即売／実演のチラシ）が印刷済みのA4厚紙**を使います。
> アプリで刷るのは **おもて面（宛名＋出店情報）** だけです。

---

## 全体の流れ

```
1. 文面をつくる  →  2. 校正に出す  →  3. ステータス更新  →  4. おもて面を印刷
（出店のご案内）   （メール/FAX/PDF）  （未着手→校正中→        （宛名＋出店情報）
                                       校正済み→印刷済み）
```

DMステータスの段階（色）：
**未着手（赤）→ 校正中（黄）→ 校正済み（青）→ 印刷済み（緑）**

---

## ステップ1：文面をつくる

1. 左メニュー **「DMハガキ」** → 右上 **「文面の作成・印刷」**。
2. **カレンダーから対象の催事をクリック**（または「検索して選ぶ」）。
   - ※出店申込書一覧の「文面へ」リンクからも、その催事を選んだ状態で開けます。
3. 文面が**ブロック（行）**で表示。各行で編集：
   - **文字サイズ**：囲み／特大／大／中／標準／小
   - **揃え**：左／中央／右
   - **上下余白**：ひろめ／標準／つめる
   - **ラベル**：行頭の見出し（例「期間」「会場」）。空なら非表示
   - **↑ ↓**＝並べ替え、**ゴミ箱**＝削除（確認あり）、**行を追加**
4. **会場（階）** は催事の「会場（階・売場）」が自動で入ります（催事編集で設定）。
5. 右のプレビュー（実寸）で確認 → **「保存」**。

### 百貨店ごとの決まりを覚えさせる（任意・おすすめ）
- 上部「**この百貨店の設定（癖・標準レイアウト）**」にメモを入力（例：会場は「○階 催事場」表記／必ず入れる文言 等）。
- 整え終えたら **「この店舗の標準として保存（文面＋メモ）」**。
- 次回、**同じ百貨店の催事を開くと自動で**この体裁とメモが出ます（店舗カルテにも表示）。

---

## ステップ2：校正に出す

文面画面の下「**校正プレビュー（実物イメージ）**」で見た目を確認してから送付。

| ボタン | 内容 |
|---|---|
| **PDFを保存（パソコンに）** | 校正用PDFを保存（保存先を選べる）。メール添付用 |
| **メールで校正依頼** | 件名・本文入りのメール下書きが開く（宛先「〇〇百貨店 DMハガキ校正ご担当者様」）。**押すと自動で「校正中」**に |
| **FAX送信状（Word）** | FAX送信状(.doc)がダウンロード→Wordで開く |
| **印刷（紙）** | 校正紙を印刷 |
| **アプリに記録（履歴）** | 送った校正PDFをアプリ保存。後で「校正履歴」から開ける |

> PCはメールに自動添付できないため、**先に「PDFを保存」→ 開いたメールに手動で添付**してください。

---

## ステップ3：ステータスを更新

- 文面画面の右上（プレビュー上）に **DMステータス**（現在＋[未着手][校正中][校正済み][印刷済み]）。
- 運用の目安：
  - **校正中**…百貨店へ校正を送付（メール送信で自動）
  - **校正済み**…百貨店からOKをもらった（印刷待ち）
  - **印刷済み**…おもて面を印刷した
- **DMハガキ一覧（`/dm`）・出店申込書一覧と連動**します。
- 一覧では**投函期限が近い未完了**の行が黄／赤＋「要対応」で目立ちます。

---

## ステップ4：おもて面を印刷する（宛名＋出店情報）

文面画面いちばん下「**宛名印刷（QR付きはがき）**」で行います。

1. **産直くんで抽出した宛名つき名簿CSV**を用意（顧客番号・氏名・住所など）。
2. CSV欄に **ドラッグ＆ドロップ**（またはクリックで選択）。
3. **列の割り当て**を確認（顧客番号・氏名は必須。自動推測されます）。
4. **「プレビューを作成」**。
5. **4面プレビュー**で仕上がり（宛名＋出店情報）を確認。
6. ズレていれば「**位置を面ごとに微調整**」（左上／右上／左下／右下の 横・縦 mm）で合わせ、**💾アイコンで保存**（この端末に記憶。次回も同じ位置）。
   - 💾がオレンジに光っているときは未保存。押すと保存され通常色に戻ります。
7. **「おもて面を印刷」** → 印刷ダイアログで **「余白なし」「等倍(100%)」** → 印刷。
   - 印刷先を「PDF」にすればPDF保存も可。

### コツ
- 用紙：**裏面チラシ印刷済みのA4厚紙**、4面割り付け → 印刷後に4分割。
- 郵便番号は **住所のすぐ上に「〒XXX-XXXX」**、住所は約65mm幅で折り返し。
- 位置合わせは **試し刷り → mm微調整 → 💾保存** が確実。一度合わせれば他催事でも使い回せます。
- 印刷後はステータスを **「印刷済み」** に。

---

## 来場登録での読み取り（当日〜後日）

- はがきおもての **QR（顧客番号）** を **USBのQR/2次元バーコードリーダー**で読み取り。
- 「来場登録」画面の番号入力欄にカーソルを置き、ピッと読む → 顧客が表示 → Enterで登録。
- リーダーは「キーボード入力＋読み取り後Enter」設定で使用。

---

## よくある質問

- **他の催事を印刷するとき**：位置設定は端末に保存済みなので**再調整不要**。名簿CSVをアップロードして印刷するだけ。
- **別のPCで印刷するとき**：そのPCで一度だけ位置を合わせて💾保存（プリンタ差があるため）。
- **裏面（チラシ）**：即売／実演の2種。**事前に厚紙へ印刷済み**の前提で、アプリでは刷りません。
- **宛名データ**：ブラウザ内で処理するだけで**保存しません**（位置設定だけ端末に保存）。
